# WordCloud-Generator
Creating a WordCloud using NLP and TF-IDF in Python

Steps:

1. Load JSON dataset in Python
2. Clean the dataset
3. Remove Encodings
4. Leemmatize the text and fit into a TF-IDF model
5. Create the World Cloud using the TF-IDF Scores.
